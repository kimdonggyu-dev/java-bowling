### 볼링 점수판

### 요구사항

- [x] 볼링 게임 점수판
    - [x] 입력
        - [x] 플레이어 이름 받기
        - [x] 투구 입력 받기
    - [x] 출력
        - [x] 프레임 구현
            - [x] 라운드 구현
            - [x] 이름 추가
            - [x] 점수 추가
                - [x] 스트라이크
                - [x] 스페어
                - [x] 미스
                - [x] 거터
- [x] 점수 계산
    - [x] 기본적으로 다음 점수에 영향을 받는다.
    - [x] 노말이면 그대로 합산
    - [x] 스페어면 10 + 다음 프레임 첫번째 스코어
    - [x] 스트라이크면 10 + 다음 프레임 첫번째 두번째 스코어
